什么是 Facebook广告库？ 

Facebook广告库的正式名称是Meta广告库。这是一个免费工具，用户可以搜索和查看任何广告主的在售和过往广告。它最初于2018年推出，主要面向与政治和社会议题相关的广告，但此后已扩展到涵盖Meta平台上的所有广告主。

Facebook推出了广告库，旨在提高用户透明度。您可以直接访问Facebook广告库：https://www.facebook.com/ads/library。

如何使用Facebook 广告库

Facebook 广告库是一个免费、全面且透明的平台，可以查看 Facebook 或 Instagram 上任何广告主的全部信息。

以下是如何使用 Facebook 广告库的方法。

打开 Facebook 广告库并搜索

第一步很简单：访问 https://www.facebook.com/ads/library进入 Meta Ad Library 。 

在 Facebook 广告库中，你会看到搜索栏。首先选择你想查看广告的国家/地区，然后从旁边的下拉菜单中选择“所有广告”。  

在搜索栏中输入商家名称（例如竞争对手的名称）或特定主题或关键词。

从那里，您将能够看到所有与您的搜索相关的结果。

点击结果

每张卡片都包含不同广告的详细信息，包括以下信息： 

  • 广告目前处于启用状态还是禁用状态？
  • 广告何时开始投放？如果已停止投放，又是何时停止投放的？
  • 广告在哪些平台（Facebook、Instagram）上投放？
  • 如果广告被标记为任何特殊类别（例如信贷、就业、住房或社会问题）

此外，在元广告库中每个广告下方，您都会看到一个“查看广告详情”按钮。点击该按钮，即可查看该广告的不同版本，包括不同的媒体、标题、描述或行动号召。 

添加筛选器

如果您想缩小搜索结果中特定元素的范围，还可以从 Facebook 广告库添加筛选器。通过筛选器，您可以按以下方式查看特定广告： 

  • 语言
  • 平台（Facebook、Instagram、Audience Network 或 Messenger）
  • 媒体（图片、表情包、图片和表情包、视频，以及无图片或视频）
  • 活动状态（活动或非活动）
  • 生效日期（过去七年内的任何时间段）

如何使用 Facebook 广告库查找竞争对手的广告

想要研究竞争对手的 Facebook 广告，最好的起点莫过于使用 Facebook 广告库。

要查看某公司在 Facebook 广告上投放的广告（如果有），您需要导航至广告库，选择竞争对手投放广告的国家/地区，然后输入其公司名称。

请注意，他们的广告很可能会链接到他们的 Facebook 页面，并且会使用相同的名称。 

从这里，您可以查看该广告商过去七年来的所有相关广告。例如，Comcast Business 的约 130 条广告都可以在这里公开查看：

然后，您可以点击每张卡片查看详细信息，或者使用筛选器查找有关竞争对手广告的更多信息。

使用这些筛选器，您可以查看竞争对手的整个创意策略，包括文字、图片和视频，还可以查看他们在过去几年特定节假日或促销期间投放的广告！ 

使用 Facebook 广告库的另外 4 个理由和方法

除了能够查看特定竞争对手的 Facebook广告之外，您还可以使用 Facebook 广告库来帮助您制定营销策略的其他方面。

寻找新思路

因为你可以按关键词或主题搜索，所以你可以使用 Facebook 广告库来激发你自己的Facebook 广告策略的新想法。

在搜索栏中，不要输入具体的品牌名称，而是输入与您的业务相关的关键词。Facebook 会从这里加载数千条其他相关企业最近创建的广告。

例如，输入“taqueria ” ，您就可以浏览数千家餐厅的真实 Facebook 广告，了解其他本地企业如何使用 Facebook 来宣传他们的特价菜、促销活动、新店地址，甚至通过 Facebook 招聘员工。

如果您刚开始使用 Facebook 广告，或者只是需要一些新的想法，这不失为一个好方法，可以了解许多其他本地企业每天都在做什么，并为您的 Facebook 及其他平台上的社交广告集思广益，提出新的想法！ 

寻找脱颖而出的机会

通过在 Facebook 广告库中搜索相关广告，您可以找到空白市场，从而帮助您的广告脱颖而出。

搜索与您的业务或竞争对手相关的广告关键词，并分析结果。您可能会发现，您所在行业许多企业的广告看起来都很相似，使用相同的语言，或者强调相同的价值主张。

利用这份分析报告来丰富你的Facebook广告，让你的广告脱颖而出。这可能意味着突出一个与众不同的优势，采用新的广告风格，或者测试能够吸引受众注意力的新词句。

分析您自己的广告

你上一次认真审视自己的Facebook和Instagram广告是什么时候？你可以使用Facebook广告库查看你企业的广告历史记录。

查看广告效果或许能发现一些优化Facebook 广告的机会。是时候更新广告素材了吗？修改广告文案？测试新的行动号召？推广新产品或服务？

就像你利用 Facebook 广告库来窥探竞争对手一样，他们也可能利用它来窥探你——所以给他们一些令人兴奋的东西看看吧！

代理商拓展

代理商在向潜在客户推介时也可以使用 Facebook 广告库。甚至在与潜在客户沟通之前，您就可以使用它来： 

检查您的潜在客户目前是否正在投放任何 Facebook 广告。

对一个完全不了解社交广告的潜在客户来说，过于专业的技术讲解可能难以理解。即使只是知道他们是否在Facebook上投放广告，也能帮助你用他们能理解的方式展开对话。如果他们正在投放Facebook广告，你就可以查看他们正在投放的广告，研究他们的主要目标、行动号召，甚至可以提供一些具体的案例，帮助他们改进Facebook广告。

查看您的潜在客户上次投放 Facebook 广告的时间。

您可能会在他们的账户中发现一些不活跃的广告——了解他们何时或最近一次暂停广告活动，可以帮助您深入了解他们面临的挑战或目标。如果您发现很多广告都集中在重要节日（例如圣诞节、夏季、返校季等）前后投放，那就说明这对他们的业务来说是一个绝佳的机会，您可以利用这一点进行推介，甚至可以提前联系他们，为他们即将开展的广告活动做好准备。

检查潜在客户或其现有代理机构的营销团队规模。

除了可以查看页面投放的广告之外，元广告库还会显示页面的负责人及其所在地。在页面名称下方，点击右侧的“关于”按钮，即可查看该商家在 Facebook 上的运营时长以及页面管理人数等详细信息。例如，这家本地正畸诊所似乎由一人管理——他可能身兼数职！ 

规模较大的公司可能在全球各地设有完整的团队或代理机构来负责其社交媒体营销活动。例如，根据 Facebook 广告库的数据，Netflix 拥有超过 200 名员工，他们来自世界各地，专门负责管理其 Facebook 页面和营销活动！

这么多人管理他们的Facebook主页，要么意味着你从一开始就需要管理多个利益相关者，要么意味着他们目前已经与一家大型代理机构合作管理他们的账户。很可能两者兼而有之！ 

利用 Facebook 广告库在竞争中脱颖而出

随着越来越多的广告主涌入 Facebook，竞争势必加剧。但 Meta 广告库可以成为 Facebook 广告生态系统中一个强大的平衡器。每天都有新的广告上传，任何精明的广告主——无论规模大小——都可以利用 Facebook 广告库密切关注竞争对手，并在 Facebook 上找到效果最佳的广告。
